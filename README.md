# 🌾 Weed Detection & Targeted Spraying (AI + Product Case Study)

##  Problem
Weed growth in wheat fields increases chemical usage and cost for farmers.

## Objective
Design an AI system for precise weed detection and targeted spraying.

## Solution Approach
- Computer Vision (U-Net + YOLO)
- Edge Computing using Jetson Nano
- On-device decision making

## Product Thinking
Focused on:
- Small-scale farmers
- Low-cost deployment
- Real-world usability

## Expected Impact
- Reduced chemical usage
- Lower operational cost
- Improved sustainability

## Key Learnings
- AI ≠ solution, usability matters more
- Constraints drive innovation
